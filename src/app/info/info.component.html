Getting Started

Article Search

The search field may be used to either search for a specific article or multiple by Title, DOI or S2PaperId. You are advised to use DOI or S2PaperId when searching for a single article.

Results Navigation

After a search you will be taken to a results page containing zero, one or multiple results which you can navigate through using the "Next" and "Previous" buttons on the bottom of the page. You can display the number of in - and out-citations and the abstract of an article using the "More Info"-button. To generate a graph, click on either the "In-citation Graph" or "Out-citation Graph" button.

Graph

When displaying a graph for the first time, you will be presented with two windows. The graph is displayed in the window to your left. You can zoom in and out of the graph by scrolling and navigate by click and drag. The panel on your right will contain "Total number of articles", which is the total number of nodes displayed in the graph. Many of these nodes may not yet contain information on the articles they represent. "Total number of articles loaded" corresponds to the total number of nodes displayed, which contain information. "Articles loaded" corresponds to the number of articles which have been fetched since you either left the results page, or since last time you pressed the "Build Graph"-button, which generates and displays the graph. The "Stop"-button ends the process of article retrieval to ensure minimal resource use - after this button is pressed you cannot continue graph building without starting over. You can click on a node to display more information on a specific article. After clicking a node you can also set it as a new seed node by clicking either "Set New Seed In-node" or "Set New Seed Out-node" (depending on whether you want to build an In-citation or Out-citation graph), followed by clicking the "Build Graph"-button. For smartphone devices, you will also be presented with two windows, but the panel will be displayed below the graph.

Settings

Here, you can change "Graph Settings". If you want to set a maximum number of children per node (i.e. maximum number of in-citation nodes or out-citation nodes for every node in a graph), you can use the number field or the slider for "Max children per node". If you want to set a maximum number of nodes for a graph, you change the value for "Max nodes", following the same approach.

Graph Definitions


Node: A scientific publication.


Current node: The node the other nodes are viewed from.


Seed node: The node from which a graph is built


In-citation node: Article that cites the current node.


Out-citation node: Paper cited by the current node.


Citation relationship: An edge between two nodes.


In-citation graph: Graph consisting of a seed node, nodes that cite the seed node and layers of nodes citing nodes in lower layers with citation relationships between nodes. Visualizes a seed nodeâ€™s impact.


Out-citation graph: Graph consisting of a seed node, nodes the seed node references and layers of nodes referenced by nodes in upper layers. Visualizes the papers the seed node depends on.
